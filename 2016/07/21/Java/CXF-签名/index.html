Cxf签名

Veröffentlicht am 2016-07-21 | Edited on 2018-10-18

生成证书

客户端

1、创建私钥和KeyStore

keytool -genkey -alias clientprivatekey1 -keypass keypass -keystore Client_KeyStore.jks -storepass storepass -dname “CN=ty.cn,C=CN” -keyalg RSA

2、给私钥自签名

keytool -selfcert -keystore Client_KeyStore.jks -storepass storepass -alias clientprivatekey1 -keypass keypass

3、导出私钥

keytool -export -alias clientprivatekey1 -file Client_PublicCert.cer -keystore Client_KeyStore.jks -storepass storepass

服务端

1、创建私钥和KeyStore

keytool -genkey -alias serverprivatekey1 -keypass keypass -keystore Server_KeyStore.jks -storepass storepass -dname “CN=ty.cn,C=CN” -keyalg RSA

2、给私钥自签名

keytool -selfcert -keystore Server_KeyStore.jks -storepass storepass -alias serverprivatekey1 -keypass keypass

3、导出私钥（将客户端公钥导入到服务端TrustStroe中，将服务端公钥导入到客户端TrustStore中）

keytool -export -alias serverprivatekey1 -file Server_PublicCert.cer -keystore Server_KeyStore.jks -storepass storepass

keytool -import -alias clientpublickey1 -file Client_PublicCert.cer -keystore Server_TrustStore.jks -storepass storepass

keytool -import -alias serverpublickey1 -file Server_PublicCert.cer -keystore Client_TrustStore.jks -storepass storepass
