Ionic3安装

安装ionic和cordova

注意：这里-g意味着全局安装

完成后，创建Ionic应用程序：

  • start 会告诉CLI创建一个新的应用程序。
  • MyIonicProject 将是您的项目中的目录名称和应用程序名称。
  • tutorial 将成为您项目的入门模板。
  • Ionic还提供以下官方模板：
    • tabs ：一个简单的3选项卡布局
    • sidemenu：侧面带有可滑动菜单的布局
    • blank：只有一页的裸露首发
    • super：拥有超过14个可用页面设计的入门项目
    • tutorial：引导式启动项目 运行您的应用程序
